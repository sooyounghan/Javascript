### while
-----
1. 조건이 참인 동안 반복 수행
2. ```https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Statements/while```
```js
let x = 0;

while(x < 10) {
  console.log(x++);
}
```

3. 무한 루프
```js
let x = 0;

while(x < 10) {
  console.log(x);
}
```

```js
while(true) {
  console.log('무한 반복');
}
```

4. break문으로 무한 반복 탈출 가능
```js
let x = 0;

while(true) {
  if(x++ >= 5) break;
  console.log(x);
}
```

5. continue와 break 사용
```js
// 1, 3, 5, 7을 의도한 코드. 그러나...
// ⚠️ 무한루프! 왜일까요?
let x = 0;
while (x < 14) {
  if (x % 2 === 0) continue;
  if (x > 7) break;
  console.log(x++);
}
```

```js
// 짧게 짠 수정 코드
let x = 0;
while (x < 14) {
  if (x++ % 2 === 0) continue;
  if (x > 8) break;
  console.log(x - 1);
}
```

```js
// 가독성 위주로 짠 코드
// 코드에 의도가 드러나도록
let x = 0;
while (x < 14) {
  const toContinue = x % 2 === 0;
  const toBreak = x > 7;
  const xNow = x++;

  if (toContinue) continue;
  if (toBreak) break;

  console.log(xNow);
}
```
